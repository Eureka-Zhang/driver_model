# 车道有效范围标定（ego_pos_y）

基于 `map/newest1-simplified.xodr` 的 OpenDRIVE 参数标定，并对应到数据列 `ego_pos_y`（单位：m）。

## 车道中心（用于理解，不建议在代码里直接写死）

- 左车道（OpenDRIVE `lane id = -1`）中心：`y ~= -3.875`
- 右车道（OpenDRIVE `lane id = -3`）中心：`y ~= -7.625`

中心点两侧每条车道半宽约 `1.875m`（车道宽 `3.75m`）。

## 有效训练范围（推荐直接用于筛选/裁剪）

### 左车道
- 标定范围：`ego_pos_y ∈ [-5.75, -2.00]`
- 训练保守范围（避开车道线，收缩 0.2m）：`ego_pos_y ∈ [-5.55, -2.20]`

### 右车道
- 标定范围：`ego_pos_y ∈ [-9.50, -5.75]`
- 训练保守范围（避开车道线，收缩 0.2m）：`ego_pos_y ∈ [-9.30, -5.95]`

## 车道分界线（两条车道的边界）
- 分界线：`ego_pos_y = -5.75`
